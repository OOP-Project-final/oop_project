## Project Description

This project performs an in-depth Object-Oriented Design (OOD) analysis of the **matplotlib** library — Python's most popular 2D data visualization library. The goal is to identify and demonstrate how real-world OOP principles (Inheritance, Encapsulation, Polymorphism, Abstraction) are applied inside the matplotlib source code.

A custom Python class `MyCustomAxis` is also built that extends matplotlib's `Axis` class to add new functionality.
